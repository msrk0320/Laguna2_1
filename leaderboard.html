Adding a run

Both builds are fully swept now, 4K–1M. To add a new quant, model, or a rebuilt fork, run the sweep unattended and fold the new rows into this page's data.

.\bench-sweep.ps1 -ContextSizes 4096,8192,16384,32768,65536,131072,262144,524288,1048576 -Label <new-label> -ServerExe "<path-to-llama-server.exe>"

Results land in bench\results.json. Paste the new objects into the DATA array at the bottom of this file's script and redeploy — the charts, stats, and table all read from that one array. 128K VRAM/RAM is missing for both current builds (monitor wasn't attached that run) — worth backfilling. Run the very first context size of a fresh session as a throwaway if you want a clean number; the OS page cache is cold until something reads the 57.6GB model file once.
